This modules converts audio signal into spikes using gamma band filtering and plots information in a single figure.

To run the function simply run script_stepbystep_modded.m n Maltab.

It uses default audio and video devices on a Windows based PC.
To disable video set useVideo to 0; Will result in a blank in one of the subplots of the main plot.
To disable spike generation set generateSpikesAudio to 0; Will result in a blank in one of the subplots of the main plot.
